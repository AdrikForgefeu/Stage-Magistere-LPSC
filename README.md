# Stage-Magistere-LPSC
Le dépôt pour le stage dans le cadre du Magistère au LPSC. Le sujet est sur la simulation d'un gaz chaud au sein d'halo galactique.

##Installation
`git clone https://github.com/AdrikForgefeu/Stage-Magistere-LPSC.git`
`pip install hot_gas`

##Création de l'environnement
`mamba env create -f environment.yml`
`mamba activate hot_gas`

##Structure
- Un dossier pour les fichiers `.tex` et `.bib`
- Un dossier pour les scripts `.py`
